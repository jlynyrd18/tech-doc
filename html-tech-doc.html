Navigation

The navigation bar on this site actually has a header above it. The bar that we want is the one that lists buy, sell, storage, ira, news, resources, and about. Without hovering over it, it could look like a standard navigation bar. When the mouse hovers over parts of the navigation bar a drop down menu appears to give mroe detailed options. Our focus is just the navigation bar and the drop down menus. The navigation bar is a list, and the drop down menus are another list within the first list. Here's an example:

Header:h3 Toppings on sandwhich h3
unodered list: ul cheese, meat, vegetables, and sauces ul
sub unordered list for cheese: ul american, peperjack, provolone, cheddar, etc. ul
sub unordered list for meat: ul turkey, ham, chicken, etc. ul
sub unordered list for vegetables: ul lettuce, spinach, peppers, olives, pickles, etc. ul

The first list is horizontal, and when hovered over one with a drop menu it has a list that drops vertically.

Slideshow

The slide show does use html and css, but also javascript. The slideshow is a very common piece of code that is used on many websites. We will be breaking down the slideshow a little bit for retention. The HTML side of the slideshow is one div class acting as a container, and each div within the container contains an image. Example:

div class="container"
div image-1 /div
div image-2 /div
div image-3 /div /div

The HTML is easier to construct once you understand it. The div tag is very common and very helpful with making containers. There is also CSS and Javascript, but CSS is just the decoration and Javascript is something I haven't learned yet.

3 Boxes

The three boxes labeled bullion made simple, exceptional service, and built for investors are lined up by making it into a flexbox. Each item uses a div tag and that turns them into a box. Flexbox then allows the boxes to be moved and scaled any way that is needed. But flexbox has very similar code in css that is required to turn it into flexbox. The basic that is needed in css is:

display: flexbox;

From there, adjustments are made to all the boxes within the container. Adjustments can be from anything to vertical alignment or horizontal, move left, right, or center. There are so many adjustments that flexbox adds, that it helps organize ecommerce websites to look a lot cleaner.

Linking

The charts are done in a backend language which I haven't learned yet, but in order to include it in a web page it is usually needed to be linked in HTML in the head part of the HTML document. It can feel frustrating at first trying to figure out how to link outside documents, but once it clicks you will know how to link to anything that is kept within the folder that holds all the documents. Within the head tag you will add a link tag that has a rel class which is short for relative, meaning what type of dicument is being linked. Then followed by the location of said document.

link rel="stylesheet" href="location of style sheet"

This is only part of what is done to get the charts todo what they do. Once someone learns javascript and begins to understand what it can do, there will be more projects that can be done to help improve your skils.

Borders

A lot like the three boxes, the three columns are able to be made thanks to flexbox. Since that would be repetitive we will focus on the borders at the top of the columns and how that can be achieved. Making a border is quite simple to make, but the variousions of it can make for some great u.i. on websites. To make a border is all done in css, and after making a couple it's easy to remember how it is made. To create a border in css, all you need to add under a tag is:

border: thickness of border, style of border, and color;

A simple one for example would be:

p { border: 5px solid black; }

That would make a solid black border that is five pixels wide around any p tag within the HTML document. Of course if you would like to have a border on just one side of the element you would add a dash then the side you want the border on. Much like this:

footer { border-top: 5px solid black; }

From my perspective they could have made a boder-top and a border-bottom in the heading of the columns. If you wish to not have corners or want a circle border instead you can add border-radius underneath the border that was just added.

Signup Forms

A basic sign up form is fairly simple to make. Forms are used a lot in HTML. The form tag wraps around the user input fields as well as label tags. Forms usually wrap around a field of user input fields that require a response like name, number, and address. It can even be used for quizzes that are online. Here's what it looks like:

form
label for="fname">First name: /label
input type="text" id="fname" name="fname"
label for="lname">Last name: /label
input type="text" id="lname" name="lname"
/form

The form itself is not visible, and also each input field is the width of twenty characters. The form above simply asks for first and last name. To set up a sign up form the type would change, and you would only need one input and label tag.

Font Awesome

Font awesome is the clickable links usually at the bottom of any webpage. They are vector images and social logos that can be just about anything, and many more are still being created. In order to use Font Awesome you will have to sign up on their website here. They are usually within an i tag or span tag. After signing up for Font Awesome they will give you a link that you must link to in your head tag of the HTML document. There are so many icons within Font Awesome that some of them you will recognize from somewhere. Typing them out would look like this:
